# SQL Task 2 - Data Insertion and Handling NULLs

## 📌 Objective

The objective of this task is to practice SQL operations such as INSERT, UPDATE, DELETE and handling NULL values in a database.

## 🛠 Tools Used

* MySQL

## 📂 Files Included

* `task2.sql` → Contains all SQL queries used in this task
* `output.png` → Screenshot of the final output

## 🚀 Steps Performed

1. Created a database named `college`
2. Created a table `students` with appropriate fields
3. Inserted multiple records into the table
4. Performed partial insertion of data
5. Updated a single record using WHERE condition
6. Updated multiple records where values were NULL
7. Handled NULL values using `IS NULL`
8. Deleted a specific record using DELETE
9. Retrieved final data using SELECT statement

## 📊 Final Output

The database contains cleaned and updated student records after performing all operations.

## 🎯 Key Concepts

* DML Operations (INSERT, UPDATE, DELETE)
* NULL Handling
* DEFAULT values
* WHERE clause

## ✅ Outcome

Successfully implemented SQL queries to manage and manipulate data, ensuring consistency and proper handling of missing values.
